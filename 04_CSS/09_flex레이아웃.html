Flex레이아웃

웹페이지 레이아웃을 효율적으로 구성하기 위한 기술, flexbox를 사용하면
요소들을 부모 컨테이너에서 유연하게 정렬하고 배치할 수 있다.

특정 요소에 display : flex -> 해당 요소를 flex 컨테이너로 사용하겠다.
flex-direction : row, column > 주 축(가로/세로) 방향 지정
justify-content : 주 축의 정렬을 어떻게 할 것인가
align-items: 교차축을 어떻게 정렬할 것인가?
(flex-start, flex-end, conter, space-around, space-between)

flexboxContainer에 flex: number -> 남는 공간을 number에 비율로 할당하겠다.
